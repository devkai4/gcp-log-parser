# GCP Log Parser

A command-line tool written in Rust for parsing and analyzing Google Cloud Logging export files.

## Features

- Parse Cloud Logging JSON exports
- Filter logs by severity, resource type, and time range
- Search log content with regex patterns
- Generate statistical reports

## Installation

### Prerequisites
- Rust 1.70 or higher

### Build from source
```bash
git clone https://github.com/YOUR_USERNAME/gcp-log-parser.git
cd gcp-log-parser
cargo build --release
```
